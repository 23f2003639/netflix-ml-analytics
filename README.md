# Netflix Analytics & Predictive Modeling (Synthetic Data)

This project showcases a data analytics and machine learning workflow using a synthetically generated dataset inspired by a subscription-based streaming platform.
The dataset is artificially created for learning and portfolio purposes. No real Netflix or user data is used.

---

## What This Project Covers

- Synthetic data generation using Python.
- Exploratory analysis of revenue, subscriptions, and marketing spend.
- Predictive modeling for business performance trends.
- Interactive Power BI dashboard for insights.

---

## Models Used

- Linear Regression (next-month price prediction).
- Random Forest Classifier (performance up/down prediction).

---

## Tools

- Python (Pandas, NumPy, scikit-learn)
- Power BI

---

## Note

This project uses synthetic data and is intended for learning and portfolio demonstration purposes only.
